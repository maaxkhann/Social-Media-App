# 📱 Flutter Social Media App

A full-featured social media app built with **Flutter** and **GetX** for state management.  

# Features
- 📝 Posts
- ❤️ Likes & 💬 Comments  
- 👥 Follow/Unfollow system  
- 💬 One-on-one Chat & 👨‍👩‍👧 Group Chat  
- 📞 Audio/Video Call functionality  
- 🔔 Real-time Notifications  

Clean architecture, Firebase backend, and responsive UI.
